Acerca de Este Sitio

Versión 1.0

Este sitio está contruido con varias tecnologías web. Esta primer versión contiene una estructura básica que permite a los participantes del grupo de La Tribu tener mayor facilidad al momento de registrarse e ingresar a obterner información de las caminatas. Por el momento dividiremos el sitio en 3 secciones para facilidad de nuestros compañer@s.

Para poder realizar esto hemos buscado la mejor tecnología que permita ser fluido, cuente con hash de seguridad o encriptadores sha256 y además de el uso de un módulo aleatorio de encriptación que consiste en convertir esa seguridad en una cifrado diferente cada vez que se ingresa al sitio, haciendo literalmente imposible predecir o bulnerabilizar cualquier contraseña protegida con esta tecnología. BCRYPT

Para ello utilizamos el lenguaje Python bajo el marco de trabajo FLASK que brinda ciertas ventajas de ligereza y portabilidad a través de un entorno virtual. Este contiene todos los paquetes y sus versiones, haciendo que aunque cada vez que exista una actualización, no provoque ningún conflicto entre librerias del sitio y por ende deje de funcionar por fallos o incompatibilidad de versiones.

Este sitio utiliza paquetes y tecnologías que respaldan proyectos muy grandes como SPOTIFY, netflix, pinterest, Nasa, google entre otros. Esto se debe a que el uso y los conectores de la base de datos sea más sencillo de administrar debido a los modelos relacionales a través sqlachemy y facilidad de migraciones mientras que la parte grafica se administra con el motor de plantillas JINJA y Flask.

En gran parte del sitio se utiliza el framework de Bootstrap o Bulma(No JS) para manera el área responsiva, Icoomon para todo lo relacionado con íconos, VUE y Javascripts para la dinámica del sitio. Cuenta además con los conectores a Mysql y Sqlite que es una base de datos pequeña para pruebas aunque no es recomentada utilizar esta última para producción. Pero no menos importante una copia de respaldo o control de versiones en trello y Github que permite ir llevando el paso a paso desde el nacimiento de este sitio web.
